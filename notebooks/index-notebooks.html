TEA Notebooks¶

This is a sequence of lectures on processing astronomical observations using Python programming, written by Jaime Zamorano, Nicolás cardiel, Segio Pascual and Jesús Gallego.

Note

These lectures were built using the new Sphinx-based Jupyter Book 2.0 tool set, as part of the ExecutableBookProject. They are intended mainly as a demonstration of these tools. Instructions for how to build them from source can be found in the Jupyter Book documentation.

These lectures are being prepared for the students of ‘Técnicas Experimentales en Astrofísica’ of Máster en Astrofísica of Universidad Complutense de Madrid
